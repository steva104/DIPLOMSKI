# VinylVibe – ASP.NET Web Application for Vinyl Sales

## Overview

ASP.NET Core MVC web application designed for online vinyl record sales.  
It allows users to browse, search and purchase records.  
The system follows a three-layer architecture and uses Entity Framework Core for database access.

## Features

- Browse and filter vinyl records  
- User registration and authentication  
- Shopping cart and order creation  
- Admin panel for product management  

## Technologies

ASP.NET Core MVC, C#, SQL Server, Entity Framework Core, Bootstrap, JavaScript, Git, GitHub

## Testing

The project includes three levels of testing:  
- **Unit tests** – implemented with NUnit and Moq  
- **Integration tests** – using an in-memory database  
- **End-to-end tests** – implemented with Playwright
